Information

Statuses:
Unsolved
Your moves so far are valid. Keep going!
Broken
There is a conflict on the board. Undo or change one or more numbers, then validate again.
Unsolvable
The board is unsolvable. Undo or change one or more numbers, then try to use solve again.
Button Actions:
Generate
Start a new sudoku table. Choose Easy, Medium, or Hard level.
Validate
Check if your current board is valid so far. If there are any conflicts, the status will change to Broken.
Solve
Automatically solve the current sudoku puzzle. Use this if you get stuck!
Meta Information
Status shows wheter your current board is Unsolved or Broken.
Level is the difficulty of the generated sudoku.
Tips
Validate anytime. It will not show where the mistake is-just whether one exists.
If you see Broken, undo a few moves or try a different path.
